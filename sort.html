BubbleSort

So funktioniert der BubbleSort

BubbleSort soll eine bestimmte Liste sortieren. Die Funktion hier schaut, ob das Zeichen nach dem aktuellen Zeichen grösser ist. Wenn nicht, dann wird das Zeichen vor das aktuelle geschoben, weil das ja grösser ist. List= [3, 1, 4, 2], Ziel= [1, 2, 3, 4] [3, 1, 4, 2] -> 1 ist kleiner als 3 -> vor 3 schieben -> [1, 3, 4, 2] [1, 3, 4, 2] -> 2 ist kleiner als 4 -> vor 4 schieben [1, 3, 2, 4] -> 2 ist kleiner als 3 -> vor 3 schieben [1, 2, 3, 4] -> richtig Algemein: Die Funktion soll eine Liste sortieren. Dabei wird die Liste von links nach rechts durchlaufen, und das Element links und das rechts miteinander verglichen. Wenn das Element rechts grösser/bzw kleiner ist, (je nach dem wie es sortiert werden soll) werden die beiden Elemente vertauscht.
